## Method of Theft

### Attack Vector: Gemini AI Trifecta Vulnerabilities

**Disclosed by Tenable Research:** September 30, 2025

**Three exploitation methods:**

1. **Cloud Assist Log Poisoning**
   - Malicious prompts injected into GCP Cloud Functions logs
   - Gemini processed hidden instructions when summarizing logs
   - Exfiltrated file contents without direct access traces

2. **Search Personalization Injection**
   - Malicious JavaScript injected into Chrome browser history
   - Gemini incorporated for personalized searches
   - Leaked user data including saved files, locations

3. **Browsing Tool Exfiltration**
   - Embedded prompts in web page content
   - Gemini routed data to attacker-controlled servers
   - Accessed via browsing integration

### Unauthorized Access Chain

```
Compromised Google Account
    ↓
Gemini AI access to Drive/Gmail/Photos
    ↓
Trifecta vulnerabilities exploited
    ↓
Private files accessed and exfiltrated
    ↓
Trade secrets stolen
```
